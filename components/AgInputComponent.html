Description

Componente AgInputComponent

El componente AgInputComponent es un control de entrada reutilizable que se puede utilizar para capturar datos de usuario en formularios. Ofrece soporte para diferentes tipos de campos de entrada, incluyendo texto, contraseñas y archivos, y proporciona funcionalidades adicionales como previsualización de imágenes y manejo de eventos de entrada y cambio de archivos.

Características principales:

  • Tipo de entrada: Permite diferentes tipos de campos de entrada como texto, contraseñas y archivos.
  • Previsualización de imágenes: Soporta la carga y previsualización de imágenes antes de su envío.
  • Eventos: Emite eventos para cambios de valor y archivos, y para la apertura de modales de búsqueda.
  • Soporte para formularios: Compatible con la interfaz ControlValueAccessor para integrarse con formularios Angular.

Uso: Este componente se puede utilizar en formularios para capturar datos del usuario, manejar la visibilidad de contraseñas y previsualizar archivos seleccionados. También es compatible con la integración de modales de búsqueda para opciones dinámicas o predefinidas.

  /**
 * Maneja el evento de entrada del campo de formulario.
 * 
 * Este método se ejecuta cuando ocurre un evento de entrada en el campo de formulario. Actualiza el valor del campo basado en la entrada del usuario y maneja la carga de archivos si se selecciona un archivo. Además, emite eventos `valueChange` y `fileChange` con el nuevo valor y el archivo seleccionado, respectivamente. Si se selecciona un archivo, también se lee y muestra una vista previa de la imagen.
 * 
 * @memberof AgInputComponent
 * @method onInput
 * @param {Event} event - El evento de entrada que contiene la información del campo de entrada.
 * @fires valueChange - Emite el nuevo valor del campo de entrada.
 * @fires fileChange - Emite el archivo seleccionado si existe.
 */
  onInput(event: Event): void {
    const inputElement = event.target as HTMLInputElement;
    if (inputElement) {
      this.value = inputElement.value;

      // Si se selecciona un archivo
      if(inputElement.files != undefined){
        const file = inputElement.files[0];
        if (file) {
          const reader = new FileReader();
          reader.onload = (e) => {
            this.imagePreview = e.target?.result;
          };
          reader.readAsDataURL(file);
          //this.value = file;
          this.valueChange.emit(this.value);
          this.fileChange.emit(file)
        }
      }
    }
  }

  /**
