Description

Componente para gestionar una paleta de widgets con funcionalidad para mostrar gráficos, tablas, formularios, alertas y otros componentes interactivos.

Este componente organiza y controla la visualización de diferentes widgets en la interfaz de usuario, incluyendo tablas, formularios, modales, y gráficos. Además, proporciona métodos para abrir y cerrar modales, manejar resultados de temporizadores y actualizar interfaces de búsqueda.
